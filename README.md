# Digital Business Card Generator

A lightweight, customizable digital business card generator built with pure HTML and CSS. Create elegant, responsive business cards that can be shared via link, QR code, or embedded on websites.

## Overview

This project allows developers and professionals to generate clean, modern digital business cards without relying on external frameworks or heavy dependencies. Built with semantic HTML5 and modern CSS3, it offers a fast, accessible solution for creating shareable professional identities.

## Features

- **Responsive Design** — Cards adapt seamlessly from mobile to desktop
- **Customizable Themes** — Easily modify colors, fonts, and layouts via CSS variables
- **Print-Ready Export** — Optimized styles for physical printing......
- **QR Code Integration** — Generate scannable codes linking to your digital card
- **Lightweight** — Zero JavaScript dependencies; pure HTML/CSS implementation
- **Accessibility First** — Semantic markup and ARIA-compliant structure
- **Social Links** — Built-in support for LinkedIn, GitHub, Twitter, and email
- **vCard Download** — Optional `.vcf` contact file generation
